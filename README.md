# Driver Route Planner Web

A simple route planning web app inspired by the referenced Driver Route Planner repository.

## Fitur

- Buat dan simpan rute
- Tambahkan titik pemberhentian dengan koordinat GPS
- Tampilkan rute pada peta interaktif OpenStreetMap
- Simpan data di localStorage

## Cara Menjalankan

1. Install dependensi:

```bash
npm install
```

2. Jalankan development server:

```bash
npm run dev
```

3. Buka browser di alamat yang muncul di terminal.

## Build

```bash
npm run build
```
